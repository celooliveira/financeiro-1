##
## Controle Financeiro V. 2
##

O Controle Financeiro esta sendo desenvolvido para você ter o controle total de suas contas, gerenciando suas despesas pessoais e tendo uma visão ampla de seus gastos de uma maneira precisa e detalhada.

##
## O sistema
##

O sistema teve inicio quando eu comecei a acompanhar minhas despesas pessoais usando uma planilha feita no excel. E vi a necessidade de ter um controle maior, resolvi criar um sistema online que atendia minhas necessidades no momento.

##
## Versão 2
##

Esta versão esta sendo utlizada a linguagem MySQLi uma extensão do MySQL mais segura e robusta e com o banco de dados reformulado, ou seja, o banco foi feito novamente desde o começo sendo criadas views e relacionamentos de chaves estrangeiras que na primeira versão não tinha.

Junto à essa versão estou utilizando o framework BootStrap 3.2
